Question-3: What is event delegation in the DOM?

Event delegation allows a parent element to handle events for its child elements using a single listener, improving performance and efficiency.


Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and propagates up to its ancestors, triggering their event handlers.
